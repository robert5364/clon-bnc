Este proyecto es una réplica visual de la interfaz del BNC que corresponde a la sección cuando los usuarios quieren acceder
a su cuenta (iniciar sesión), creado exclusivamente para practicar maquetación web (HTML/CSS) y lógica de componentes (JS).

### Disclaimer
Este proyecto es una réplica visual con fines exclusivamente **educativos**. No tiene relación oficial con el
Banco Nacional de Crédito (BNC) y no recopila, almacena ni solicita datos de usuario.

No debe ser utilizado para ningún otro propósito que no sea la revisión de código por parte de reclutadores o estudiantes.
